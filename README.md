# Line Highlighter HTML

Resalta líneas con etiquetas HTML específicas (h2, dialog, etc.) con colores configurables.

## Configuración

Edita los settings en Pulsar para añadir/quitar etiquetas y colores.

## Instalación

1. Descarga el ZIP
2. En Pulsar: Settings > Install > Install from .zip file
3. Reinicia Pulsar

## Uso

- Abre un archivo HTML/PHP
- Las líneas con h2, h3, etc. se resaltarán automáticamente
- Activa/desactiva los elementos a resaltar  en Settings > Packages > line-highlighter-html
- Configura colores en Settings > Packages > line-highlighter-html
